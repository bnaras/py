Important

This is a simple web application that does not have all the bells and whistles you'd want to make it usable for the general public. So, you've been informed.

  • You should do this only once each quarter. Redoing it will move the database and all the details on current requests etc. making them not easily usable.
  • Ensure that the time slots have been completely finalized. There is no way to change them over the web (although it is easy to do so on the command line.)
  • You need a list of enrolled people with their sunet ids and roles. You can additional people later through this webapp if needed, but you cannot remove those already added.
